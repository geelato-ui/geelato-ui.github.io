# 介绍

Geelato UI是基于Vue Ant Design的扩展，提供企业后台应用中常用到的页面及组合，快速实现CRUD等管理。引入页面组件、卡片式布局、JSON化配置、自研的GQL，结合内置的后台常规API，便于实现灵活的界面设计器，实现轻代码化开发。

Geelato UI仍然处于开发中，这里有一些目前还不支持、但已经在计划中的特性：

  • 页面布局
  • 列表
    • 查询列表
    • 树列表
  • 表单
  • 工具条
  • 弹出窗口
  • API
    • 登录、登出
    • CRUD
    • 组件加载
    • 配置页面加载保存

我们欢迎你为Geelato UI的开发作出贡献。
